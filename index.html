DIY Satellite Ground Station

SatNOGS is an open source ground station, optimized for modularity, built from readily available and affordable tools and resources.

Learn more

A work in progress.

Part of NASA's SpaceAppsChallenge 2014, we set off to create a solution for the Ground Station Platform challenge. We are moving fast so keep checking this page!
